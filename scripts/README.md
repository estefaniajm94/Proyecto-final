# Scripts de apoyo

Utilidades locales para acelerar tareas de desarrollo y validacion manual.

- `paste_clipboard_to_emulator.ps1`: envia el texto actual del portapapeles de Windows al campo enfocado de un emulador Android listo.

Ejemplo de uso:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\paste_clipboard_to_emulator.ps1
```
